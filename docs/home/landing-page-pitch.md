# AI-MDE Landing Page Pitch

## Hero
**AI-MDE**  
**Model-Driven Engineering rebuilt for AI**

## Subheading
Turn requirements into architecture, models, and build-ready specs through a command-driven, rule-enforced engineering pipeline.

## Supporting line
AI-MDE replaces ad-hoc prompting with explicit commands, reusable skills, architecture policies, and deterministic artifact generation.

## Value Props

### 1. Explicit Commands
Tell the system what phase to execute:
- Validate Requirements
- Build System Design
- Generate LDM
- Generate Module Spec

### 2. Architecture Enforcement
Generation is constrained by:
- layer rules
- module type rules
- DTO policy
- data conventions
- concurrency rules

### 3. Traceable Artifacts
Every output fits into a governed pipeline:
requirements → architecture → model → modules → specs → validation
